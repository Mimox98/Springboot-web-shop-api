Welcome to my Web Shop API

Explore and test the core resources: Category, User, and Product. All endpoints are plain REST with JSON.

Resources & CRUD

Each controller exposes Create, Read, Update, and Delete operations.

Category

CRUD
  • GET /categories — list categories
  • POST /categories — create category
  • GET /categories/{id} — get by id
  • PUT /categories/{id} — update
  • DELETE /categories/{id} — delete

User

CRUD
  • GET /users — list users
  • POST /users — create user
  • GET /users/{id} — get by id
  • PUT /users/{id} — update
  • DELETE /users/{id} — delete

Product

CRUD

Before creating a product, make sure there is at least one Category in the system.

  • GET /products — list products
  • POST /products — create product (requires category)
  • GET /products/{id} — get by id
  • PUT /products/{id} — update
  • DELETE /products/{id} — delete
